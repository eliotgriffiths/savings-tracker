Why iOS and iPadOS can't support this: Apple requires all browsers on iOS and iPadOS to use Safari's engine. Even if you install Chrome or Edge, they are running on top of Safari underneath. Since Safari doesn't support the File System Access API, no browser on these devices can offer local file auto-save.

Alternatives that work everywhere: Use Network Location / NAS or a cloud service — these use standard web requests and work on every browser and device.

Encryption

Password-encrypts your backup files using AES-256. A recovery key file is downloaded when you set a password — keep it safe. Without it, a forgotten password means your data cannot be recovered.

Set backup password

Your backup files will be encrypted with this password.

Important: If you forget this password and lose your recovery key, your data cannot be recovered. There is no reset option.
A vault-recovery.json file will be downloaded automatically. Store it somewhere safe — it can unlock your backup if you forget your password.
